# Your Prototype

This app was created from the PM Lab template. Auth and database are pre-configured.

## Local development

1. Install dependencies: `pnpm install`
2. Pull env vars: `vercel env pull .env.local`
3. Run dev server: `pnpm dev`
4. Open http://localhost:3000

## Deploying

Push to `main` to deploy to production. Open a PR for a preview deployment.

## What's included

- **Authentication**: Email/password signup and login via Supabase Auth
- **Database**: Shared Postgres with your own schema namespace, accessed via Drizzle ORM
- **Dashboard**: Example CRUD interface for Items and Comments
